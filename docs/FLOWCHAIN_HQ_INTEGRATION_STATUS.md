# FlowChain HQ Integration Status

This public file intentionally avoids duplicating live issue, PR, branch, and worktree state.

Use GitHub issues, pull requests, milestones, and Actions as the source of truth for current coordination. Historical internal worktree matrices and prompt-pack paths are not committed to the public launch repository because they expose stale implementation plans and local coordination details.

## Public Boundary

- This repository may document accepted architecture, reproducible local/test commands, public release gaps, and reviewed decision records.
- It should not expose raw agent goal packs, local worktree paths, operator host details, private research archive locations, or internal launch-run artifacts.
- If a coordination decision becomes durable, promote it into `docs/DECISIONS/` or the relevant public guide with tests or checks named.
