# Agent Bonds Public Claim Architecture

Public-claim generation is now structured and gated.

Claim levels:
- internal_dev
- capped_pilot
- integration_beta
- public_protocol_available
- broad_public_launch

Broad public launch remains blocked by default.

Blocked phrases include:
- risk-free
- guaranteed safe
- insured
- fully insured
- guaranteed payout
- guaranteed reimbursement
- first ever
- audit-approved
- unlimited public launch
- uncapped

Passport/Envelope/Receipt readiness is required before integration-beta and above should be described as enabled.
