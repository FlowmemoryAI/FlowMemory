# Agent Bonds A2A Integration

Agent Bonds does not replace A2A.

A2A remains the agent-discovery and agent-to-agent task/delegation surface.

FlowMemory adds:
- a FlowMemory Agent Bonds Agent Card extension;
- bonded-task envelope metadata;
- receipt references for completed or disputed work.

Public A2A cards must not include:
- secrets;
- private evidence URIs;
- internal admin addresses.

The A2A extension is gated by Passport/Envelope/Receipt readiness.
