# Unsorted Inbox

This is the existing raw-intake area for FlowMemory ideas, prompts, references, and half-formed notes that are not yet accepted architecture or scoped work.

Use this folder when an input is useful enough to preserve but not ready for `research/`, `docs/DECISIONS/`, an issue, or implementation.

## Allowed here

- Raw product, protocol, research, hardware, and agent-memory ideas.
- External project links and short source notes.
- Prompt fragments and operator thoughts that need later sorting.
- Open questions and “smallest useful next step” notes.

## Not allowed here

- Secrets, seed phrases, private keys, RPC credentials, API keys, or webhook URLs.
- Heavy model artifacts, embeddings, media, generated datasets, or evidence bundles.
- Accepted architecture decisions.
- Production claims, tokenomics commitments, public-validator plans, mainnet claims, or audited-cryptography claims.

## Promotion rule

A note can leave this folder only after it has:

1. a clear owner or work area;
2. a bounded scope;
3. a distinction between off-chain data and on-chain commitments;
4. a verifier/indexer reconstruction story; and
5. explicit speculation and non-goal boundaries.

Until then, every file here is intake, not project commitment.
